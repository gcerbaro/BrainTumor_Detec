# Detector de Tumores Cerebrais

## Como executar

### Ambiente Python + Dependências
```bash
$ make
```

### Servidor (API)
```bash
$ make api
```

### Cliente
Em outro terminal, executar o seguinte comando:
```
$ make client
```

## Como acessar

A aplicação deve estar acessível no endereço: http://localhost:3000
